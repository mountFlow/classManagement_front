# 课室排课信息管理系统前端
## 1.Project setup
```
npm install
npm run serve
```
## 2.项目简介

- 前后端分离开发并实现了基础的课室排课系统
- 分为教务员和学生两类账号
- 前端使用elementUI来构建所有表单界面
- 后端有完备的mysql数据库
- 通过mybatis来读取，
- 通过axios传值
- 自写排课算法。

技术栈:vue、vue-cil、elementUI、axios、Java mybatis、servlet
